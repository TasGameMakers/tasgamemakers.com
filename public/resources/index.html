Resources

A collection of tools, funding sources, learning materials, and communities useful for game makers in Tasmania and beyond.

Game Engines

Unity Popular cross-platform game engine for 2D and 3D games. → Unreal Engine Powerful engine for high-fidelity 3D games and experiences. → Godot Free and open-source engine with its own scripting language. → Bitsy A tiny editor for little games or worlds—no coding required. → Twine An open-source tool for telling interactive, nonlinear stories. →

Art & Design

Aseprite Animated sprite editor and pixel art tool. → Blender Free and open-source 3D creation suite. → Krita Free digital painting application for concept art and illustrations. →

Audio

Audacity Free, open-source audio editor and recorder. → FMOD Adaptive audio middleware for games. → freesound Collaborative database of Creative Commons licensed sounds. →

Funding

Screen Tasmania Tasmanian government screen funding and support. → Screen Australia Australian federal screen funding body with games programs. → Film Victoria Victorian screen funding body supporting games development. →

Learning

GDC Vault Talks and presentations from the Game Developers Conference. → Game Dev Beginner Tutorials and guides for those starting out in game development. → Brackeys Popular YouTube channel with Unity and game dev tutorials. →

Community

TasGM Discord Our own Discord server—join the conversation! → IGDA International Game Developers Association. → Freeplay Australia's longest-running independent games festival. →

Supporting Tasmanian creatives. We're inclusive, creative, passionate about making games, and proudly Tasmanian.
